# Understanding Systems with C

This repository documents my journey of learning C as a foundation for understanding how computer systems actually work under the hood.

Instead of focusing only on syntax, the goal here is to explore how programs interact with memory, how data is represented and manipulated at a low level, and how fundamental system concepts are built using C.

The work in this repository focuses on building a deeper understanding of how low-level programming concepts come together to form the foundation of computer systems.

This is not just about writing programs — it’s about building a mental model of how software interacts with hardware.

Over time, this repository will evolve as I dive deeper into low-level programming and system design, forming a strong base for more advanced areas like embedded systems and hardware security.

---

**Focus:** Learning by building, experimenting, and understanding — not just coding.
